# QuestReward

A structure for a reward given after completing a [Quest](./Quest.md)

## Properties
 - rewardID - ID of the [Skill](./Skill.md) or [Item](./Item.md) to add or remove
 - ammount
 - rewardType = "xp"; // (skill), (xp), (item), (remove)
 - redeem()
