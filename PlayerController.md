# PlayerController

Manage the inputs a player would put in

# Controls
 - WASD - move
 - 1-9 - select spell :: maybe they can have QA bindings as well. 1:Q, 2:E, 3:R, 9:'space'
 - LeftMouse - cast spell
 - RightMouse - interact
 - LeftALT - quick stat sheet / Debug?
 - TAB - inventory/CharacterSheet/Quests/Spellbook
