# Speaker

Anything/Anyone with this Speaker module is interactable.
Interacting with this object will send [Dialogue](./Dialogue.md) to the [Dialogue Manager](./DialogueManager.md)

## Uses
 - [Dialogue](./Dialogue.md)


## Properties
 - SpeakerName - Passed in on init. This is usually the name of the Character/Object
 - DefaultResponse List - this is loaded from a file somewhere based on SpeakerName

### Dialogue options:
 - Default response list
